# Roadmap

## Now: Private Alpha Desktop

- Make the desktop app testable end to end.
- Keep AURA local-first and approval-first.
- Strengthen AURA Guardian as the visible trust layer.
- Make memory useful, scoped, editable, and safe.
- Make workflow replay safe and explainable.
- Keep launch flows visible from the command center.

## Next: Reliability

- Sign/notarize packaged Mac builds.
- Add embedded or guided Python runtime setup.
- Improve live browser automation hardening.
- Add richer UI tests for approval cards and Guardian events.
- Add crash/log reporting that does not leak secrets.

## Later: Platform Expansion

- Phone approval companion.
- User-owned encrypted cloud backup.
- Enterprise identity/policy server.
- Team workflows and audit export.
- Home/car/wearable adapters after the desktop trust core is mature.

## Not Now

- Payment processing.
- Cloud account requirement.
- Random new consumer features unrelated to context, safety, memory, cost, workflows, identity, tools, or audit.
